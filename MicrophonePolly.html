Local RTC Connection (by Swolekat)

Allow local programs to interface with this UI programmatically. They will be able to send the following messages:

  • setup sets aws region, identity pool, and audio device
  • start starts the speech recognition
  • stop stops the speech recognition
  • randomVoice chooses a random voice
